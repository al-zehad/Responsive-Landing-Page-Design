# Responsive Landing Page Design

This is a **fully responsive landing page** built using **HTML and Tailwind CSS**. The project focuses on modern layout structure, clean spacing, and responsive behavior across all devices. It was completed as part of an assignment from **Ostad**, a live learning platform in Bangladesh.

### The project

This project is a **responsive landing page design** created using raw HTML and Tailwind CSS utility classes. The goal was to practice building scalable layouts, managing spacing, and ensuring consistent responsiveness across different screen sizes as part of a structured learning assignment.
